# vuecli-project

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```


### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
we use jest, you need to write .spec.js in @tests/unit
```

### if mock
```
vue.config.js 删除proxy，且main.js中删除import './mock'
```
